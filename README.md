# ProyectoColaborativoPython

## Descripción del proyecto
Proyecto simple compuesto por módulos que demuestran el uso de funciones en Python. Incluye operaciones matemáticas, un cuento y utilidades para texto.

## Descripción de los módulos

**matematicas.py**  
Módulo con operaciones matemáticas básicas: suma, resta, multiplicación, división (con control de error por cero), potencia y residuo.

**cuento.py**  
Módulo que imprime el cuento clásico "El conejo y la tortuga" junto con su moraleja sobre la constancia y el esfuerzo.

**utilidades.py**  
Módulo con funciones útiles para texto: convertir a mayúsculas, a minúsculas, contar caracteres e invertir texto.

**main.py**  
Archivo principal que importa todos los módulos y muestra ejemplos de su uso.

## Instrucciones de uso

1. Descarga o clona todos los archivos del repositorio en la misma carpeta.
2. Asegúrate de tener Python 3 instalado.
3. Ejecuta el programa con el siguiente comando:

```bash
python main.py
